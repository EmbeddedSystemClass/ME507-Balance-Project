Detailed Description

The main file for a ME507 balancing device.

Contains the declarations to globally accessible pointers to the shared data items and queues which are used to exchange information between tasks; code which creates the pointers to the shares and queues; and the function main(), which is the first function to run when the program is started up. Inside main(), code does initial setup, creates the task objects which are used in the program, and starts the RTOS scheduler.

Function Documentation

int main ( void  )

This function runs when the application is started up.

The main() function instantiates shared variables and queues, sets up serial ports and other drivers (if needed), creates the task objects which will run ad infinitum, and starts the RTOS scheduler.

Returns
Although the scheduler never exits and this function never returns, it's C/C++ tradition for main() to return an integer, so we (don't) return zero; if the return statement is missing the compiler issues a warning.
